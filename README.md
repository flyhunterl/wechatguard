# 微信守护程序

## 功能特点

- 🛡️ 微信窗口守护
- 🕒 系统空闲时间检测
- 🔐 密码保护设置
- 🖥️ 系统托盘图标管理

## 环境要求

- Windows 10/11
- Python 3.8+
- 管理员权限

## 使用方法

### 启动程序

- 双击 `start.bat`（以管理员权限运行）

### 系统托盘菜单

- 开始守护：进入守护模式
- 停止守护：退出守护模式
- 设置：配置密码和空闲时间
- 帮助：查看使用说明
- 退出：关闭程序

## 注意事项

- 请始终以管理员权限运行
- 尊重他人隐私
- 仅在合法和有道德的情况下使用
